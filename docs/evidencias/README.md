# Evidencias do Projeto

Use esta pasta para guardar registros que comprovem o desenvolvimento do SmartWaste.

## O que registrar

- Fotos da modelagem ou preparo no Bambu Studio.
- Fotos da impressao na Bambu Lab A1.
- Fotos das pecas impressas antes da montagem.
- Fotos do prototipo montado.
- Prints do Monitor Serial do ESP32.
- Prints do dashboard web.
- Resultados de entrevistas ou formularios, sem dados pessoais sensiveis.

## Convencao de nomes

```text
bambu-studio-lixeira-v01.png
impressao-a1-tampa-sensor-v01.jpg
prototipo-montado-v01.jpg
serial-leitura-vazia-v01.png
dashboard-alerta-critico-v01.png
```

## Cuidados

- Evite imagens com senhas, chaves de API ou dados pessoais.
- Prefira fotos nítidas e bem iluminadas.
- Inclua data ou versao no nome quando houver varias tentativas.
