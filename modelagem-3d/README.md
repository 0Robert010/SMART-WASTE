# Modelagem 3D SmartWaste

Esta pasta organiza os arquivos usados para criar, fatiar e imprimir as pecas do prototipo fisico da lixeira inteligente.

## Ferramentas

- Impressora 3D: Bambu Lab A1.
- Software: Bambu Studio.

## Estrutura

```text
modelagem-3d/
|-- fontes/
|-- exports/
`-- referencias/
```

## Convencao de nomes

Use nomes curtos, descritivos e versionados:

```text
lixeira-corpo-v01.3mf
lixeira-tampa-sensor-v01.3mf
suporte-esp32-v01.3mf
suporte-leds-v01.3mf
canaleta-cabos-v01.3mf
```

Quando uma peca for aprovada para impressao final, salve uma versao nova em vez de sobrescrever a anterior.
